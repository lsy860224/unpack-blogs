## 발행 절차 (편당 ~45분)

### 1. 스마트에디터 열기
네이버 블로그 → 글쓰기 → **스마트에디터 ONE**

### 2. 에디터 기본 설정
- 카테고리: 해당 에디션 메타 참고
- 공개설정: 전체공개
- 댓글·공감: 허용
- 검색 허용: ON

### 3. 제목 복붙
각 파일 "── 제목 ──" 아래 한 줄

### 4. 본문 복붙
"────────── 본문 복붙 시작 ──────────" ~ "본문 끝" 사이

**주의:**
- Markdown 볼드(`**...**`)는 네이버 에디터에서 **굵게 자동 변환되지 않을 수 있음**. 수동으로 `Cmd+B` 굵게 지정
- 소제목은 **더 크게(본문2/본문3)** 포맷으로 별도 지정
- `[이미지 삽입]` 자리표시자 → 실제 이미지 업로드

### 5. 이미지 배치 (10장)
파일 상단 "이미지 10장 배치 플랜" 표 참고.

**이미지 크기 조정:**
- 네이버 에디터 "원본" 업로드 → 자동 중앙 정렬
- 이미지 아래에 **캡션** 한 줄 추가 권장 (체류시간↑)

### 6. 태그 입력
각 에디션 메타의 태그 10개 복붙 (쉼표 구분)

### 7. 발행

### 8. 발행 직후 (3분)
- [ ] 발행된 URL 복사
- [ ] 네이버 서치어드바이저 → **요청 → 웹페이지 수집** → URL 붙여넣고 [확인]
- [ ] Obsidian Pipeline 카드 업데이트:
  - `naver_edition_status: published`
  - `publish_url_naver: <url>`
  - `publish_date_naver: YYYY-MM-DD`

## 이웃 소통 루틴 (발행일)

발행 후 2시간 내:
1. 이웃 신규글 Top 5~10개 공감 눌러주기
2. 댓글 달린 이웃 블로그 답방
3. 공감 눌러준 이웃에 "이웃이벤트" 메시지 (선택)

이 루틴이 **네이버 C-Rank·D.I.A. 알고리즘**에 활발함 신호를 줍니다.

## 3편 발행 후 기대값

첫 주 현실적 목표:
- 방문자: 30~100명
- 공감: 편당 5~10개
- 댓글: 편당 1~3개
- 이웃 신규 유입: 10~20명

숫자 적어도 **체류시간·댓글 소통**이 알고리즘에 더 중요합니다.
