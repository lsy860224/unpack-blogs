# 비개발자가 AI로 iOS 앱 만든 후기 — Claude Code로 2주 만에 App Store 출시

────────── 본문 복붙 시작 ──────────

**"Flutter를 한 줄도 써본 적 없는데, 2주 만에 App Store에 iOS 앱이 올라갔습니다."**

2026년 4월 22일 기준, 제가 만든 앱 **GentleDo**가 App Store에 공식 출시됐어요. 투입 시간은 **약 45시간**. 혼자서, 퇴근 후 시간에만.

[대표 이미지 삽입]

---

비결은 **Claude Code**라는 AI 코딩 도구였습니다.

부업으로 앱 만드시려는 분들께 솔직하게 공유할게요. **AI 코딩으로 어디까지 되고, 뭐가 안 되는지** 실전 후기입니다.

---

## **Claude Code가 뭔가요?**

터미널에서 돌리는 **AI 코딩 에이전트**입니다. ChatGPT·Cursor랑 비슷하지만 차이가 있어요.

- **ChatGPT** — 대화형. 코드 복붙해서 질문
- **Cursor** — IDE 안에서 블록 단위 AI 제안
- **Claude Code** — **터미널에서 파일 직접 읽고·쓰고·커밋**

가장 큰 차이는 **"커밋 전체를 조율하는 감각"**이에요. "이 기능 만들자" 한 문장이면 Claude Code가 **파일 5개를 자동으로 고치고 테스트까지 돌립니다.**

[Claude Code 터미널 스크린샷 삽입]

---

## **비용은 얼마였나요?**

2주간 실제 투입한 돈입니다.

- Claude Code Max 구독 — **월 $20** (약 27,000원)
- Apple Developer Program — **연 $99** (연 1회, 월 환산 8,250원)

**월 35,000원 정도**로 앱 하나를 만들었어요. 외주 견적으로 Flutter 앱 개발 맡기면 최소 300만원부터 시작하는데, 말이 안 되는 가성비죠.

[월 비용 정리 표 이미지 삽입]

---

## **왜 Flutter를 골랐을까?**

비개발자 1인에게 Flutter가 가장 현실적이었어요.

- ✅ **iOS + Android + macOS 한 코드베이스** — SwiftUI랑 Kotlin 따로 배울 수 없음
- ✅ **Dart의 정적 타입** — AI 페어 프로그래밍할 때 실수가 적음
- ✅ **Hot Reload** — 저장하면 0.5초 내 반영. 피드백 루프가 짧음

단점도 있어요.

- ⚠️ iOS 네이티브 기능은 패키지 품질에 의존 (나중에 실제로 크래시 났어요)
- ⚠️ 기본 위젯 그대로 쓰면 "Flutter로 만든 티" 남

---

## **2주간 제가 뭘 했고, AI가 뭘 했나**

정직하게 말씀드리면, Claude Code가 혼자 다 한 게 **아닙니다.**

### **제가 한 일 (사람의 영역)**

- 프로젝트 컨텍스트 문서화 (CLAUDE.md 규칙집)
- 실기기 테스트, 디자인 결정, 카피라이팅
- "이건 지금 하지 마세요" 리스트 유지
- 가설 정의와 검증 지표 설정

### **Claude Code가 한 일 (AI의 영역)**

- Dart 코드 실제 작성
- DB 스키마 마이그레이션 코드
- 상태관리(Riverpod) 설계
- 반복 태스크 스케줄링 로직
- 테스트 케이스 작성
- 리팩토링 제안

[내가 한 일 vs AI가 한 일 표 이미지 삽입]

## **핵심은 CLAUDE.md 파일이었어요**

프로젝트 루트에 `CLAUDE.md`라는 **규칙 문서**를 쓰는 게 전부였습니다.

이 문서가 없으면 AI는 매번 조금씩 다른 스타일로 코드를 짜요. 있으면? **같은 지시를 두 번 할 일이 사라집니다.**

제가 쓴 규칙 예시:

- Red 컬러 사용 금지
- UI 문자열 하드코딩 금지 (다국어 파일 경유 필수)
- Phase 1에서 서버·AI API 추가 금지
- Material 아이콘 대신 lucide_icons만

이 4줄이 수백 번의 **"그거 말고"** 소통을 절약했어요.

[CLAUDE.md 파일 스크린샷 삽입]

## **45시간을 어디에 썼나?**

투입 시간 내역을 솔직히 공개합니다.

- **20시간** — 평일 야간 개발 (2시간 × 10일)
- **10시간** — 주말 개발 (5시간 × 2일)
- **7시간** — 실기기 디버깅 (특히 크래시 잡는 데 시간 많이 씀)
- **8시간** — Xcode 설정·스토어 제출·메타데이터·스크린샷

**총 ~45시간**. 시급으로 환산하면 외주 견적 대비 말도 안 되는 효율이에요.

[45시간 내역 표 이미지 삽입]

---

## **솔직히, AI로 안 됐던 것 3가지**

과장 없이 공개합니다. 바이브 코딩 한계도 알고 시작해야 해요.

### **1️⃣ 실기기 디버깅**

시뮬레이터는 멀쩡한데 실기기에서 크래시 나는 버그가 있었어요. **Claude에게 크래시 로그 줘도 원인 못 찾았습니다.** 결국 제가 GitHub 이슈 트래커 뒤져서 해결.

### **2️⃣ Xcode 설정의 체크박스 하나**

TestFlight 암호화 선언, iPad 타깃 제거, provisioning profile 설정. **여기서 하루를 날렸어요.** Xcode UI는 AI가 못 봅니다.

### **3️⃣ 디자인 감각**

"이 카드가 좀 무거워 보인다" 같은 피드백은 텍스트로 표현이 안 돼요. 결국 Figma에서 직접 조정한 뒤 값만 AI에게 옮겼습니다.

---

## **그럼 부업으로 정말 가능한가요?**

**가능합니다. 단 조건이 있어요.**

✅ **가능한 조건:**
- 월 35,000원 구독 비용 감당 가능
- 주 10~15시간 투입 가능
- 기술 용어 20개 정도는 읽을 줄 알아야 (Widget, State, Provider 등)
- 로컬 전용·단순 MVP 범위에서 시작

❌ **이런 사람은 권하지 않음:**
- 서버·인증·결제 시스템이 핵심인 앱 (난이도 급상승)
- 기술 문서 읽기 자체가 힘든 경우
- 앱 아이디어 없이 "그냥 만들어보고 싶다" 수준

---

## **GentleDo는 어떤 앱인가요?**

에너지에 맞춰 할 일을 제안하는 **생산성 앱**입니다.

아침마다 컨디션 체크 한 번 → 그날 에너지에 맞는 할 일만 필터링해서 보여줘요. 피곤한 날엔 가벼운 일만, 좋은 날엔 도전적인 일을.

- **iOS 전용** (무료, 로컬 저장)
- **한국어·영어 지원**
- **광고·계정 없음**

👉 App Store에서 **"GentleDo"** 검색하시거나, 아래 링크로 바로 받으실 수 있어요.

🔗 [App Store에서 GentleDo 받기](https://apps.apple.com/us/app/gentledo/id6761796867)

## **AI 코딩 도구, 뭐부터 시작할까?**

AI 도구 처음 쓰시는 분께 추천 순서:

1. **ChatGPT 무료** — 개념 질문·디버깅 시작점 (부담 없음)
2. **Claude Code Max $20** — 본격 개발 시작 시 (가성비 최강)
3. **Cursor Pro $20** — IDE에서 작업하고 싶으면

처음엔 ChatGPT 무료로 감 잡고, 프로젝트 시작할 때 Claude Code 전환이 현실적이에요.

## **마무리**

**"1인 개발로 앱 만드는 시대가 왔다"** 같은 표현을 많이 보셨을 거예요. 뜬 구름 잡는 이야기 같지만, 제가 45시간으로 출시까지 간 게 증거입니다.

완벽한 앱은 못 만들어요. 하지만 **"혼자서는 못 만들었을 앱"**을 **"혼자서도 만들 수 있게"** 해주는 도구는 확실합니다.

부업으로 앱 하나 만들어보고 싶으셨다면, 지금이 시작하기 좋은 타이밍이에요.
