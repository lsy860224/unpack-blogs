# 네이버 에디션 글쓰기 가이드

> babipa의 AIGrit — AIGrit 본사이트의 생활·재테크 프레임 에디션 + 협찬 전용

## 정체성

- 블로그명: babipa의 AIGrit
- 성격: AIGrit 리프레이밍 + 네이버 전용 협찬
- 타겟: AI 도구에 관심 있는 한국 직장인·부업러
- 수익: 애드포스트 + 협찬·체험단
- **본사이트 글과 완전히 다른 글처럼 보여야 한다**

## 리프레이밍 규칙 (핵심)

네이버 독자와 구글 독자는 **전혀 다른 사람**. 같은 글 복붙은 양쪽 다 망친다.

### 제목

| AIGrit 본사이트 | 네이버 에디션 |
|---|---|
| Claude vs ChatGPT — 블로그 글쓰기 실전 비교 | 퇴근 후 AI 글쓰기 1주일 실험 후기 |
| Apple 단축어 AI 자동화 5가지 | 월 5만원 절약하는 아이폰 단축어 5개 |
| Make 입문 — 반복 업무 자동화 | 퇴근시간 1시간 벌어주는 자동화 툴 |

**프레임:** "퇴근 후~", "월 N만원~", "N일 실험", "직장인 필수~", "초보도 가능~"

### 본문 비교

| 요소 | 본사이트 | 네이버 |
|---|---|---|
| 첫 문단 | 역피라미드 (결론 선제시) | **스토리 훅** ("3일 전 내가 겪은 일") |
| 소제목 | H2/H3 시맨틱 | **볼드 텍스트 (크게)** |
| 문단 길이 | 4~6줄 | **2~3줄** (모바일 가독성) |
| 이미지 | 3~5장 | **10장+** (체류시간) |
| FAQ | 필수 3~5개 | **생략** 또는 본문에 녹임 |
| CTA | 제휴링크 | **이웃추가·공감·댓글** 유도 |
| 내부 링크 | 5~7개 | 외부 URL 최소 (네이버 선호도 낮음) |

## 콘텐츠 공급

### 방식 1: AIGrit 에디션 (메인)

1. AIGrit 본사이트 글 발행 완료
2. Session 2 (별도 탭)에서 `blog-naver` 스킬 + naver_edition_angle
3. Craft `01. In Progress/Naver/` 저장
4. 이미지 10장+ 확보
5. 네이버 스마트에디터 복붙 발행
6. 소요: 약 1시간

### 방식 2: 네이버 전용 협찬 글

- AIGrit 본사이트에는 게재하지 않음
- 공정위 문구 필수
- 솔직한 단점 1개 이상

## 이미지 10장+ 확보

| 소스 | 수량 |
|---|---|
| 본사이트 재활용 | 3~5장 |
| 추가 스크린샷 | 2~3장 |
| 다이어그램·요약 | 1~2장 |
| 밈·반응 이미지 | 1~2장 |
| 구분선 템플릿 (재사용) | 2~3장 |

## 네이버 SEO

- 제목: 핵심 키워드 앞배치, 30자 이내
- 첫 100단어: 키워드 3회 반복
- 소제목: 볼드 텍스트 키워드 강조
- 태그: 5~10개
- 체류시간: 이미지 10장+, 문단 짧게
- 카테고리 일관성
- 네이버 서치어드바이저 수동 수집 요청
