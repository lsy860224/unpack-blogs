## 2. ❌ Search Advisor 등록 불가 (확정)

- `blog.naver.com/aigrit` 호스트는 **네이버 소유** → Search Advisor 사이트 등록 거부
- 네이버 블로그 관리자에 **"검색 반영 요청" 메뉴도 존재하지 않음** (2026-04-25 확인)
- 글별 수동 인덱싱 요청 경로가 공식적으로 없음

### 대신 할 수 있는 것
1. **검색 허용 ON** 한 번만 확인: `기본 설정 → 기본 정보 관리 → 블로그 정보`에서 "검색 허용 ✓ + 외부 공개 ✓"
2. **Search Advisor `간단체크` 탭**에 URL 입력 → 색인 상태 확인 (등록과 다름)
3. **기다리기** — 보통 수 시간 ~ 24h 내 자동 반영

## 3. C-Rank 가중치 요인 (추정)

| 요인 | 가중 | 실천 방법 |
|---|:---:|---|
| **체류시간** | 높음 | 이미지 8장+, 문단 2~3줄, 구분선 다수 |
| **페이지 이동** | 높음 | 프롤로그 이미지 강조형 + 목록형 (여러 글 탐색) |
| **이웃 수** | 중 | 초기 이웃 20~30 확보, 댓글 적극 |
| **공감·댓글** | 중 | CTA 명시 ("공감 ♡ 댓글 한 번만…") |
| **카테고리 일관성** | 중 | AI 도구 리뷰 · 생산성 자동화 · 1인 개발자 — 3~4개 이내 |
| **키워드 첫 100단어 반복** | 중 | Primary 3회, 자연스럽게 |
| **이미지·영상 존재** | 낮 | 이미지 8장+ 표준 |
| **백링크** | 낮 | 네이버는 외부 링크 가중치 낮음 |

## 4. D.I.A. (Deep Intent Analysis) 대응

네이버의 **의도 기반 랭킹 알고리즘**. "검색 의도와 얼마나 맞는가"를 본다.

### 의도 매칭
- **제목에 사용자 고민 표현** 포함 — "퇴근 후 30분", "월 5만원 절약"
- **첫 문단 = 검색 고민 답** — "평일 저녁 9시. 설거지 끝내고 남은 건 30분…"
- **카테고리·태그 정합성** — 글 내용과 정확히 일치

### 실천
- AIGrit 본사이트 글과 다른 검색 의도 타깃 — 본사이트: 도구 비교 / 네이버: 부업·절약·직장인

## 5. 키워드 전략

### 첫 100단어 키워드 3회 반복
```
"회사 업무만으로도 벅찬데 부업 준비는 언제 하지?"
평일 저녁 9시. 설거지 끝내고 아이 재우고 나니 남은 건 30분. 이 짧은 시간에 부업 기반을 만들 수 있을까 고민했습니다.
답은 Notion AI였어요. 월 14,000원 투자로 하루 30분이 실제 생산성으로 바뀌었습니다.
```
→ "Notion AI" + "부업" + "생산성" 3회 자연 반복.

### 네이버 전용 연관 키워드
- 직장인 + 퇴근 후 + 월 N만원 + 초보 + 꿀팁 조합
- LSI 키워드(잠재 의미): 스마트에디터가 자동 인식

## 7. 카테고리 일관성

```
📁 공지·일상 (2)           ← 인사, 자기소개
📁 AI 도구 리뷰 (10~15)    ← 핵심
📁 생산성 자동화 (5)       ← 에디션
📁 1인 개발자 이야기 (5)   ← babipanote 교차
📁 협찬·리뷰 (월 2~4)      ← 전용 협찬
```

C-Rank는 **같은 카테고리 내 연속 발행**을 가산점으로 인식 (추정). 카테고리 돌아다니지 말고 2~3주 단위로 한 카테고리 집중.

## 8. 프롤로그 · 홈 레이아웃 (C-Rank 핵심)

### 필수 세팅 (최초 1회)
- **대표 메뉴: 프롤로그** (포스트형 금지)
- 프롤로그 레이아웃: **이미지 강조형**
- 블로그 목록: **목록형** 또는 **앨범형** (한 페이지 5글)

### 이유
- 방문자 **페이지 이동 횟수** = C-Rank 가중 (AIGrit이 알고 있는 유일한 강한 신호)
- 썸네일 중심 레이아웃 → OG/프롤로그 이미지 투자 회수

상세: [image/NAVER.md](../image/NAVER.md) §3.

## 9. 이웃 & 공감 신호

### 초기 이웃 확보 (Week 1~2)
- 이웃 20~30명 신청 (동일 카테고리 활동 블로거 위주)
- 이웃 댓글에 답글 (알고리즘 신호)
- 공지글 1개 발행 (블로그 정체성 명시)

### 공감·댓글 유도 (글 단위)
- 글 하단 고정 CTA: "공감 ♡ 댓글 한 번만 남겨주시면 제작에 큰 힘이 됩니다 🙏"
- 질문 훅: "여러분은 어떻게 쓰세요?" 오픈 엔드 마무리

## 10. 수익화 (애드포스트 + 협찬)

### 네이버 애드포스트
- 신청 조건: 글 5개+ 누적
- 예상 수익: 월 ₩1,000 ~ ₩30,000 (초기)
- 지급 조건: 누적 ₩5,000+

### 협찬·체험단
- 플랫폼: 레뷰·디너의여왕·미스터블로그 → 방문자 3,000+ 후
- 직접 제안: 방문자 5,000+, 이웃 500+
- 단가: 글당 ₩100,000 ~ ₩500,000

### 공정위 문구 (협찬 시 필수)
```
본 콘텐츠는 {브랜드}로부터 원고료·제품을 제공받아 작성되었으며,
실제 체험 기반 솔직한 후기입니다.
```
본문 **최상단**에 배치.
