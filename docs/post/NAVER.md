## 2. 핵심 원칙: 리프레이밍

### 제목 프레임

| AIGrit 본사이트 | 네이버 에디션 |
|---|---|
| Claude vs ChatGPT — 블로그 글쓰기 실전 비교 | 퇴근 후 AI 글쓰기 1주일 실험 후기 |
| Apple 단축어 AI 자동화 5가지 | 월 5만원 절약하는 아이폰 단축어 5개 |
| Make 입문 — 반복 업무 자동화 | 퇴근시간 1시간 벌어주는 자동화 툴 |
| Notion AI 완벽 활용법 | 퇴근 후 30분, Notion AI로 부업 기반 만들기 |

**프레임 옵션**: `"퇴근 후 ~"`, `"월 N만원 ~"`, `"N일 실험 ~"`, `"직장인 필수 ~"`, `"초보도 가능 ~"`

**제약**: 30자 이내, 핵심 키워드 앞배치.

### 본문 리프레이밍

| 요소 | 본사이트 | **네이버** |
|---|---|---|
| 첫 문단 | 역피라미드 | **스토리 훅** ("3일 전 내가 겪은 일") |
| 소제목 | H2/H3 시맨틱 | **볼드 텍스트 크게** (스마트에디터) |
| 문단 길이 | 4~6줄 | **2~3줄** (모바일 가독성) |
| 이미지 | 3~5장 | **8장+ 패키지 표준** (체류시간) |
| FAQ | 필수 3~5개 | **생략 또는 본문에 녹임** |
| CTA | 제휴 링크 | **이웃추가·공감·댓글** 유도 |
| 내부 링크 | 5~7개 | **외부 URL 최소** (네이버 선호도 낮음) |

## 3. 본문 구조

1. **스토리 훅** (2~3문장) — "평일 저녁 9시. 설거지 끝내고…"
2. **답/주제 제시** — "답은 X였어요."
3. **본문 섹션** — "활용 #1", "활용 #2" 형식 또는 날짜·단계형
4. **비교표** (선택) — "팀 vs 혼자 쓸 때"
5. **FAQ** (선택, 5문항 이내) — 본문 녹임 실험 권장
6. **마무리 + 다음 에디션 예고**
7. **이웃추가·공감·댓글 CTA** 반드시

---

## 4. 콘텐츠 공급 방식

### 방식 1: AIGrit 에디션 (메인)
1. AIGrit 본사이트 글 발행 완료
2. Session 2 (별도 탭)에서 `blog-naver` 스킬 + `naver_edition_angle` 사용
3. Craft `01. In Progress/Naver/` 에 초안 저장
4. 이미지 8장 패키지 확보 ([image/NAVER.md](../image/NAVER.md))
5. 네이버 스마트에디터 복붙 발행
6. 예상 소요: **1시간**

### 방식 2: 네이버 전용 협찬 글
- **AIGrit 본사이트에는 게재하지 않음** (리뷰 객관성 혼란 방지)
- Craft `[Naver-협찬] 제품명` 별도 문서
- **공정위 문구 본문 상단 필수** ("본 콘텐츠는 해당 브랜드로부터…")
- **솔직한 단점 1개 이상** 포함
- 거절 리스트 유지 (브랜드 가치 훼손 시)

## 5. 절대 금지

- 본사이트 글 **복붙** — 양쪽 중복 콘텐츠 → 양쪽 SEO 타격
- 공정위 문구 누락 (협찬 시) — 법적 리스크
- "실패" / "깨짐" / "포기" 등 부정 표현 (GentleLab 원칙 일부 적용)
- 외부 링크 남발 (네이버 C-Rank 신호 불리)
- FAQ 섹션 남발 (네이버 독자는 본문 흐름 선호)
- "포스트형" 레이아웃 유지 (→ 프롤로그·목록형 전환 필수)

## 7. 발행 워크플로우

1. **본사이트 AIGrit 원본** 발행 완료 확인
2. **Craft Naver 초안** 작성 (리프레이밍 규칙 적용, 마커 포함)
3. **이미지 패키지** 8장 준비 + 프롤로그용 `00-thumb-square.png` 생성 ([image/NAVER.md](../image/NAVER.md))
4. **`/review-post`** — 네이버 기준 체크리스트로 검증 ([SEO](../seo/NAVER.md))
5. **수정** (이미지 마커 정합, 태그 정합, 발행 메타 블록)
6. **스마트에디터 발행**:
   a. Craft에서 Cmd+A → Cmd+C 복사 (이미지는 따라오지 않음)
   b. 스마트에디터 붙여넣기
   c. 이미지 8장 재업로드 (패키지 폴더 순서대로)
   d. **대표 이미지 `00-thumb-square.png` 설정** (프롤로그용)
   e. 카테고리 + 태그 5~10개 입력
   f. 발행
7. **발행 후**:
   - Obsidian Pipeline 카드 `naver_edition_status: published`
   - `publish_url_naver: https://blog.naver.com/aigrit/{logNo}`
   - Craft 문서 → `02. Archive/Naver/` 이동
   - AIGrit 본사이트 Pipeline 카드에 역참조 기록
   - **인덱싱 수동 요청 불가** — C-Rank 자동 인덱싱 ([seo/NAVER.md](../seo/NAVER.md))
