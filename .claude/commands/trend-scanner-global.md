# /trend-scanner-global — 글로벌 시장 트렌드 탐색

글로벌 시장 부업/사업 트렌드를 탐색하고 기회를 분석합니다.

## 실행 지시

사용자의 입력에서 타겟 시장(미국/유럽/동남아/전체)과 도메인을 파악하세요. 명시 없으면 미국/영어권 중심.

다음 순서로 진행하세요:

1. **영어 소스 중심 웹 서치**로 최소 5개 소스에서 데이터 수집
   - Statista, Gartner, TechCrunch, Indie Hackers, Product Hunt 등
2. 트렌드 최소 3개 도출 — 각각: 요약, 성장 지표(USD), 기회 영역, 규제/세무 고려사항, 진입 장벽(USD), 타이밍, 한국인 적용 시 고려사항(세금/언어/결제), 글로벌 레퍼런스
3. 기회 최소 5개를 6가지 기준(시장 매력도, 진입 용이성, 수익 잠재력, 경쟁 강도, 확장성, 규제 안정성)으로 /30 스코어링
4. 핵심 리스크 요약(규제 + 한국 세무 유의사항: 해외소득 신고, 이중과세, 영세율) 포함
5. 마크다운 리포트로 산출 — USD 기준, 필요시 원화 환산 병기
## 산출물 끝에 반드시 포함

### 3관점 검증
- 🔴 비판적: 가장 약한 근거, 한국인 적용 시 현실적 장벽, 과대 추정 가능성
- 🟢 긍정적: 가장 강한 기회, 글로벌 확장성, 시장 성장 근거
- ⚪ 중립적: 추가 확인 필요한 데이터, 가정의 신뢰도
