# /brand-architect — 브랜드 아이덴티티 통합 설계

네이밍 → 보이스 → 비주얼(컬러/타이포/로고) → 디지털 프레즌스 → 가이드라인 + JSX 프리뷰를 생성합니다.
모듈별 독립 실행 가능: "이름만" "컬러만" 등으로 범위 지정 가능.

## 실행 지시

1. **네이밍**: 후보 10개 생성 → 도메인/상표(KIPRIS)/SNS 핸들 웹 서치 검증 → 5가지 기준 가중 스코어링 → 최종 3개
2. **브랜드 보이스**: 퍼스낼리티 5축 스펙트럼 + 톤앤매너 + 호칭/금지 표현 + UVP + 태그라인 3개 + 메시지 기둥 3~5개
3. **비주얼 아이덴티티**:
   - 컬러 팔레트: 5색 시스템(Primary/Secondary/Accent/Neutral Dark/Light) + 라이트&다크 모드 쌍 + HEX/RGB/HSL + WCAG 대비율 체크 + 경쟁사 컬러 차별화
   - 타이포: 한글+영문 페어링(무료 폰트 우선) + 6단계 스케일(Display~XSmall)
   - 로고: 유형 선택 + 디자인 브리프(키워드/금지/아이콘 방향) + 사이즈 사양표
4. **디지털**: OG 이미지 규격 + 메타 태그 HTML 코드 + SNS 프로필/배너 규격 + 파비콘
5. **가이드라인 문서**: 마크다운으로 전체 통합 + CSS 변수 + Tailwind Config
6. **JSX 프리뷰**: brand-preview.jsx 파일 생성
   - 라이트/다크 토글, 컬러 스와치, 타이포 렌더링, 로고 플레이스홀더, 버튼/카드/배지 UI 샘플, 미니 LP 히어로, OG 시뮬레이션, 보이스 예시
   - Tailwind + CDN 폰트 로드 + default export
## 산출물 끝에 반드시 포함

- 🟢 긍정적: 브랜드 차별화의 강점, 타겟에 대한 감성적 연결, 확장성
