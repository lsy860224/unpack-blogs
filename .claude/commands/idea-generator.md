# /idea-generator — 부업/사업 아이디어 생성 & 스코어링

트렌드 스캔 결과 또는 사용자 키워드에서 구체적 아이디어를 생성하고 검증 로드맵까지 설계합니다.

## 실행 지시

입력 유형을 판별하세요:
- A) 이전 trend-scanner 산출물 기반 → 상위 기회에서 아이디어 파생
- B) 사용자가 직접 주제 제시 → 해당 주제에서 생성
- C) 범위 없음 → 관심 도메인 먼저 확인

다음 순서로 진행하세요:

1. 웹 서치로 관련 사례/수익 데이터 리서치
2. **최소 7개 아이디어** 생성 — 각각에 10가지 요소 정의:
   아이디어명, 한 줄 설명("누구를 위해/무슨 문제를/어떻게"), 타겟 고객, 핵심 가치 제안, 수익 모델, 예상 가격대, MVP 범위(3~5기능), MVP 구축 기간, 경쟁 상황, 레퍼런스
3. 7가지 기준(문제 절실도, 지불 의사, 시장 크기, 구축 난이도, 경쟁 우위, 확장 가능성, 방어 가능성)으로 /35 스코어링
4. **상위 3개에 30일 검증 스프린트** 설계 — Week 1(문제 검증) → Week 2(솔루션 검증) → Week 3(MVP) → Week 4(유료 전환)
5. 개인 컨텍스트(기술/시간/자본/열정)가 있으면 적합성 필터링 표시
## 산출물 끝에 반드시 포함

### 3관점 검증
- 🔴 비판적: 가장 실패 확률 높은 아이디어와 이유, 스코어링에서 과대평가된 항목, 검증 없는 가정
- 🟢 긍정적: 가장 유망한 아이디어의 핵심 강점, 시장 타이밍이 좋은 이유
- ⚪ 중립적: 스코어링 점수의 신뢰도, 추가 검증이 필요한 가정 목록
