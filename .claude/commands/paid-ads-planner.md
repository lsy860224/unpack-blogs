# /paid-ads-planner — 유료 광고 플랜

채널 선정 → 예산 배분 → 크리에이티브 → ROAS 시뮬레이션을 설계합니다.

## 실행 지시

1. **컨텍스트**: 제품 유형, 타겟 프로필, 월 예산, 광고 목표(인지/트래픽/전환)
2. **채널 선정 매트릭스**:
   - 한국: 네이버 검색/쇼핑검색/GFA, 카카오 비즈보드/모먼트, 당근마켓, Meta, 구글, 유튜브 — 적합 업종/최소 예산/과금 방식/특징
   - 글로벌: Google/Meta/X/LinkedIn/TikTok/Reddit
   - **인플루언서**: 나노(1K~10K)/마이크로(10K~100K)/미들/매크로 — 단가+ROI
3. **예산 배분**: 채널별 비율/예산/예상 클릭/전환/CPA/ROAS 테이블 + 예산 단계별 권장안(30만~300만+)
4. **크리에이티브**: AIDA/PAS 프레임워크 + 채널별 카피안 3개 변형 + 스펙(글자 수, 영상 길이)
5. **첫 2주 테스트 플랜**: 테스트 변수, 성공 기준, 스케일업 기준
6. ROAS < 1이면 즉시 중단 권고. 광고법 준수 주의.
## 산출물 끝에 반드시 포함

### 3관점 검증
- 🔴 비판적: ROAS 예측의 약한 가정, 예산 소진 위험, 크리에이티브 피로도
- 🟢 긍정적: 채널 선택의 타겟 매칭, 테스트 전략의 효율성
- ⚪ 중립적: CPC/CPM 단가의 변동 가능성, 벤치마크 데이터 신뢰도
