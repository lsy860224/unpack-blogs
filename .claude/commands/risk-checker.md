# /risk-checker — 리스크 분석 & 대응 전략

5대 카테고리(규제/세무/시장/운영/개인) 리스크를 식별하고 매트릭스로 평가합니다.

## 실행 지시

1. **웹 서치**로 업종 관련 규제/법률 최신 정보 확인
2. **5대 카테고리 체크리스트** 점검:
   - A. 규제/법률: 사업자등록, 인허가, 개인정보보호법, AI 기본법, 전자상거래법, 지적재산권
   - B. 세무/재무: 소득 유형, 부가세, 종합소득세, 해외소득 신고, 4대보험
   - C. 시장/경쟁: 시장 축소, 대기업 진입, 기술 변화, 고객 집중도
   - D. 운영/기술: 1인 번아웃, 기술 부채, 외부 서비스 의존, 보안 사고
   - E. 개인/생활: 본업 겸업 금지, 번아웃, 생활비 확보
3. **리스크 매트릭스**: 각 리스크를 발생 확률(1~5) × 영향도(1~5)로 평가 → 점수 15~25=상, 8~14=중, 1~7=하
4. 최소 7개 리스크 식별. "리스크 없음" 결론은 금지.
5. 🔴 상 리스크에 **회피/완화/전가/수용** 대응 전략 + 구체적 액션
6. 규제/인허가 체크리스트 + 세무 요약 + 주요 신고 일정 포함
## 산출물 끝에 반드시 포함

### 3관점 검증
- 🔴 비판적: 발견하지 못한 리스크가 있는가, 대응 전략이 현실적인가, 규제 변화 가능성
- 🟢 긍정적: 대부분의 리스크가 관리 가능한 수준인가, 경쟁사도 같은 리스크를 가지는가
- ⚪ 중립적: 규제/법률 정보의 최신성, 리스크 평가의 객관성
