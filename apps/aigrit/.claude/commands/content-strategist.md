# /content-strategist — 콘텐츠 마케팅 전략 & 캘린더

메시징 프레임워크 → SEO/AIO → 플랫폼별 전략 → 3개월 캘린더를 설계합니다.

## 실행 지시

1. **메시징 프레임워크** (모든 콘텐츠의 기반):
   - 원라인 UVP, 엘리베이터 피치(3문장), 핵심 키워드 3개, 톤앤매너, 메시지 기둥 3~5개, 페르소나별 변형
2. **키워드 & 토픽**: 웹 서치로 시드→롱테일 확장 + 토픽 클러스터(필러+클러스터) + 검색 의도(정보/비교/구매)
3. **AIO 전략**: AI 검색 최적화 — 질문-답변 포맷, 구조화 데이터, E-E-A-T 강화
4. **플랫폼별 전략**:
   - 한국: 네이버 블로그/카페, 인스타(릴스+캐러셀), 유튜브, 크몽/탈잉, 브런치 — 각각 콘텐츠 유형/포맷/빈도/핵심 전략
   - 글로벌: 블로그(SEO), X, LinkedIn, Product Hunt, Reddit
5. **리퍼포징 플로우**: 1개 원본 → 캐러셀/쇼츠/뉴스레터/쓰레드/카톡 변환
6. **3개월 캘린더**: 주차별 요일·플랫폼·토픽 배치 (가용 시간에 맞춰 현실적)

부업자 기준 주 3~5시간 콘텐츠 제작. 품질 > 수량.
