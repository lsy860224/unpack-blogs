## 챗봇 — Claude와 ChatGPT, 뭐부터 써야 할까?

한국어로 긴 글을 쓸 거라면 Claude, 구조적 답변이 필요하면 ChatGPT다. 두 개 다 유료로 쓰는 게 결론이다. 월 $40이 아깝지 않다.

<Link href="/ko/blog/claude-4-sonnet-vs-gpt-4o">Claude 4 Sonnet vs GPT-4o 비교</Link>에서 자세히 다뤘듯이, 같은 프롬프트로 블로그 초고 10개씩 써봤을 때 Claude 쪽 문장 흐름이 더 매끄럽다. 반면 코드 주석·API 문서 생성은 ChatGPT가 아직 더 정돈돼 있다. Gemini는 2026년 들어 Google 워크스페이스 연동이 강해졌지만, 한국어 문체만 놓고 보면 3위다.

## AI 검색 — Perplexity는 구글을 대체할까?

정답부터: **대체는 아니고 분업이다.** 구글은 쇼핑·지도·동영상에 그대로 쓰고, 정보 조사와 출처 확인에만 Perplexity를 쓴다.

<Link href="/ko/blog/perplexity-ai-guide">Perplexity AI 사용법</Link>에 자세히 적었지만, Perplexity의 핵심은 "답변 아래 출처 링크 5~10개"다. ChatGPT가 그럴듯한 답을 만들어 내도 출처를 내 손으로 검증해야 한다면, Perplexity는 검증이 끝난 채로 온다. 블로그 글 리서치 시간이 절반으로 줄었다.

단점도 있다. 최신 뉴스는 Google보다 20~30분 늦다. 그리고 한국어 전용 정보는 네이버 지식인이 아직 이긴다.

## AI 코딩 — Claude Code vs Cursor, 뭘 고를까?

**터미널 중심 작업은 Claude Code, IDE 중심은 Cursor.** 1인 개발자라면 둘 다 쓰는 걸 추천한다.

<Link href="/ko/blog/claude-code-vs-cursor">Claude Code vs Cursor 비교</Link>에서 한 달 병행 사용 결과를 정리했다. 실제 프로젝트에서 역할은 이렇게 갈렸다:

- **Claude Code** — Git 조작, 모노레포 파일 이동, 빌드 스크립트 자동화
- **Cursor** — 컴포넌트 리팩토링, 타입 에러 수정, 테스트 코드 작성

<Link href="https://babipanote.com/blog/building-app-with-claude-code">babipanote의 Flutter 앱 3개 빌드 기록</Link>에서 Claude Code가 비개발자 워크플로우에 어떻게 들어오는지 더 솔직하게 썼다.

## 지식관리 — Craft vs Notion · Obsidian의 자리

지식관리 도구는 "문서 작성용"과 "지식 축적용"으로 분리해야 한다.

<Link href="/ko/blog/craft-vs-notion">Craft vs Notion 비교</Link>에서 다뤘듯이, Craft는 "글 쓸 때 집중력이 좋은 종이 같은 UI"가 강점이고 Notion은 "팀 협업·데이터베이스"가 강점이다. 1인 빌더는 Craft, 소규모 팀은 Notion이 맞다.

Obsidian은 이 두 개와 다른 층이다. 마크다운 원본을 로컬에 두고 관계 그래프로 지식을 축적하는 용도. 블로그 글쓰기는 Craft, 장기 지식 축적은 Obsidian. 역할이 겹치지 않는다.

## 무료 도구만으로도 충분할까?

결론부터: **입문 1~2개월은 무료로 충분하고, 3개월차부터는 유료가 이득이다.**

무료 조합의 한계는 명확하다. Claude 무료 플랜은 일일 메시지 한도가 있고, ChatGPT 무료는 GPT-4o 사용 제한이 있다. 한 세션만 길게 써도 끊긴다. 글 한 편 쓰는 도중 "한도 초과" 알림이 뜨면 집중력이 깨진다.

그래도 무료로 시작하고 싶다면 이 조합이다:
- **Claude 무료** — 하루 10~15번 대화
- **Google Gemini 무료** — Google 계정만 있으면 무제한
- **Perplexity 무료** — 일일 5회 Pro 검색
- **Apple 단축어** — 완전 무료

직장인은 이 조합으로 2~3개월 체감해 본 뒤, 자기 사용 패턴이 잡히면 유료 1~2개만 골라 가는 게 합리적이다.

1. **Jasper** — 마케팅 카피 생성기. Claude로 대체 가능. 월 $49 아쉽다.

공통점: **"기능이 특화돼 있지만 범용 챗봇이 점점 잘해진다"**다. 2026년 AI 도구 선택 기준은 "좁고 깊게"가 아니라 "한 개 강력한 챗봇 + 불가피한 특화 도구"로 굳어졌다.

## 실제 지출 공개 — 구독료 검증

구독료는 꼭 정기 점검하자. 나는 3개월마다 한 번씩 "최근 30일간 열어본 횟수"를 체크한다. 5회 미만이면 해지 후보다. 이 점검 방식 덕에 월 $80에서 $55까지 내려왔다.

실제 지출 사례가 궁금하다면 위에서 링크한 수익·비용 리포트를 보자. 월 59,000원으로 AI 도구 4개 + 블로그 2개 운영까지 커버한 실측 기록이다.

## 자주 묻는 질문 (FAQ)

### AI 도구 초보자는 뭐부터 써야 하나요?

ChatGPT 무료 버전으로 시작하세요. 일주일 써보고 "내가 장문 글쓰기가 많다" 싶으면 Claude Pro $20, "구조적 답변·코드가 많다" 싶으면 ChatGPT Plus $20로 넘어가는 게 최소 실수입니다. 처음부터 둘 다 결제할 필요 없습니다.

### 한국어 품질은 어느 게 제일 좋나요?

2026년 4월 기준 Claude > ChatGPT > Gemini 순서입니다. 특히 장문 글쓰기·문체 유지·한국어 고유 표현에서 Claude가 앞섭니다. 다만 코드 주석·API 문서는 ChatGPT가 여전히 우세합니다. 용도별로 다르게 쓰는 게 답입니다.

### 기업용 엔터프라이즈 플랜은 어떻게 고르나요?

10인 이하는 Claude Team($25/시트)이나 ChatGPT Team($30/시트)이 가성비 좋습니다. 50인 이상은 Enterprise 협상이 유리합니다. 법무·보안 요구가 있으면 Anthropic·OpenAI 세일즈와 직접 협의해 데이터 보관·로깅 정책을 계약서에 명시하는 걸 추천합니다.

### AI 도구 몇 개가 적당한가요?

매일 여는 건 3~5개가 한계입니다. 10개를 결제해도 실제로 매일 켜는 건 3~4개로 수렴합니다. 구독 후 30일 뒤 사용 로그를 체크해서 주당 1회 미만이면 해지하세요. 도구 수를 늘리기보다 "한 도구를 깊게 쓰는 것"이 생산성에 더 큰 영향을 줍니다.

### 학생 할인이나 무료 크레딧 있나요?

GitHub Student Pack에 Notion·Figma·JetBrains 등이 포함돼 있고, Claude와 ChatGPT는 별도 학생 할인이 없습니다(2026-04 기준). Midjourney는 학생 할인이 없지만 연간 결제 시 $10 → $8 수준 할인 혜택이 있습니다. 학생은 무료 Claude + Gemini + Perplexity 조합으로도 충분합니다.

### 개인정보 보호가 걱정됩니다.

Claude·ChatGPT 모두 기본 설정에서는 대화가 학습에 사용될 수 있습니다. 설정에서 "학습용 데이터 제공 끄기"를 체크하거나, API 버전을 쓰세요(API는 기본적으로 학습에 쓰이지 않음). 완전히 로컬에서 돌리고 싶다면 Ollama + Llama 3 조합을 고려해 보세요. 속도는 느리지만 데이터가 기기 밖으로 나가지 않습니다.
