**결론부터: Perplexity는 출처 있는 답을 주는 AI 검색엔진이지만, 구글을 완전히 대체하진 못한다. 리서치용으로 쓰면 시간을 반으로 줄인다.**

3주간 구글 대신 Perplexity만 써봤다. 논문 조사, 기술 비교, 뉴스 확인, 쇼핑 검색 — 모든 검색을 Perplexity로 돌렸다. 결론은 "완전 대체는 아니지만, 특정 용도에서는 구글보다 확실히 낫다."

## Perplexity는 구글과 무엇이 다를까?

구글은 **링크 목록**을 준다. Perplexity는 **답변 + 출처**를 준다. 차이가 간단해 보이지만 워크플로우에서는 크다.

| 항목        | Google             | Perplexity            |
| ----------- | ------------------ | --------------------- |
| 결과 형태   | 링크 10개          | 요약 답변 + 출처 링크 |
| 팔로업 질문 | 새 검색 필요       | 대화형 추가 질문      |
| 출처 신뢰도 | 사용자가 직접 판단 | AI가 출처 명시        |
| 광고        | 상단 3~4개         | 없음                  |
| 한국어      | 완벽               | 영문 대비 80%         |
| 속도        | 즉시               | 5~10초 (AI 처리)      |

## 무료 기능으로 할 수 있는 것

무료 계정으로도 하루 5~10회 Pro 검색 가능. 일반 검색은 무제한. 충분히 써볼 수 있다.

**무료로 강한 영역:**

- 기술 문서 요약 (공식 docs가 너무 길 때)
- 최신 뉴스 확인 (출처 포함 요약)
- 비교 리서치 ("A vs B 차이점")
- 코드 에러 해결 (Stack Overflow 대체)

## 실제 사용 로그 — 3주 기록

### 1주차: "오, 이거 편하다"

- 기술 비교 검색에서 구글 대비 체감 50% 시간 절약
- 팔로업 질문이 자연스러워서 "검색 → 클릭 → 읽기 → 다시 검색" 루프가 사라짐

### 2주차: "한국어가 좀..."

- 한국어 검색 품질이 영문 대비 떨어짐 (특히 네이버 블로그 소스 반영 약함)
- 한국 법률·정책 관련 검색은 구글이 압도적

### 3주차: "병행이 답이다"

- 리서치·비교·기술 문서 = Perplexity
- 로컬 정보·쇼핑·한국 뉴스 = 구글
- 두 개를 번갈아 쓰는 게 최적

## 한계와 대안

**Perplexity가 못하는 것:**

- 이미지 검색 (구글 압승)
- 지도·로컬 검색 (구글 맵스)
- 한국어 커뮤니티 검색 (네이버 카페·블로그)
- 실시간 주가·환율 (전문 앱이 나음)

**대안 비교:** <Link href="/ko/blog/claude-4-sonnet-vs-gpt-4o">Claude 4 Sonnet vs GPT-4o 비교</Link>에서 다룬 것처럼, 챗봇으로 검색하는 것과 Perplexity로 검색하는 것은 용도가 다르다. 챗봇은 "생성", Perplexity는 "검색+요약". 둘을 헷갈리면 쓸 일도 아닌 곳에 쓰게 된다. AI 도구 전체 지형 속에서 Perplexity의 위치는 <Link href="/ko/blog/ai-tools-2026-guide">2026 AI 도구 완벽 가이드</Link> Pillar에서 "검색" 카테고리로 묶었다. 도구를 "하나로 통합" 하려는 시도가 왜 실패하는지는 <Link href="/ko/blog/craft-vs-notion">Craft vs Notion</Link>에서도 같은 맥락으로 다뤘다.

자동화 워크플로우 측면에서는 <Link href="/ko/blog/apple-shortcuts-ai-automation">Apple 단축어 AI 자동화</Link>에서 소개한 "한 줄 요약" 단축어와 Perplexity 모바일 앱을 조합하면 모바일 리서치 속도가 또 한 단계 올라간다. 전체 자동화 스택에서 Perplexity 의 위치는 <Link href="/ko/blog/solo-developer-automation-stack">1인 개발자 자동화 스택</Link>에서 "리서치 전용 검색" 으로 정리해뒀다. 공식 사이트는 [perplexity.ai](https://www.perplexity.ai).

## 자주 묻는 질문 (FAQ)

### Perplexity는 무료인가요?

무료 + Pro $20/월 이중 구조다. 무료로도 일반 검색 무제한, Pro 검색 하루 5~10회 가능. 대부분 무료로 충분하다.

### 출처는 믿을만한가요?

대체로 정확하지만 교차 검증은 필요하다. 특히 한국어 소스는 네이버 블로그 등 신뢰도가 낮은 출처를 인용하는 경우가 있다. 학술 목적이면 원문 확인 필수.

### 한국어 잘 되나요?

영문 대비 약 80% 수준. 기술·IT 주제는 양호하지만, 한국 문화·법률·정책 관련은 구글이 낫다. 개선 속도는 빠르다.

### ChatGPT 검색과 뭐가 다른가요?

ChatGPT는 대화·생성 중심에 검색을 붙인 구조. Perplexity는 검색·인용 중심에 대화를 붙인 구조. 리서치가 목적이면 Perplexity가 출처 관리 측면에서 더 일관적이다.

### Perplexity Pro 학생 할인이 있나요?

주기적으로 파트너 프로모션이 뜬다 (통신사 번들, 학생 인증 등). 공식 사이트 Pricing 페이지를 수시 확인하면 좋다.

## 마무리

Perplexity는 "AI 검색엔진"이라는 포지션을 제대로 소화하는 거의 유일한 도구다. 구글을 대체한다는 프레임보다, **"리서치 전용 보조 검색"** 이라는 프레임으로 접근하면 가치가 명확해진다. 무료로 2주 써본 뒤 Pro 여부를 결정해도 늦지 않다.
